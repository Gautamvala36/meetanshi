  • WYSIWYG editor to set a custom text, related images or any other HTML code for success or failure message of applying the coupon code.

  • The "Manage Coupon Code Links" backend grid enlists all the coupon code links in the store. If the new coupon codes have been created,the admin has to click "Update Coupon Code Links" to update the link list.
